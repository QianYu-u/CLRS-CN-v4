# 前言

不久前，任何听说过"算法"这个词的人几乎肯定是计算机科学家或数学家。然而，随着计算机在我们现代生活中变得无处不在，这个术语已不再神秘。如果你环顾自己的家，你会发现算法在最平凡的地方运行：你的微波炉、洗衣机，当然还有你的计算机。你要求算法为你做出推荐：你可能喜欢什么音乐或驾驶时应该走哪条路线。我们的社会，无论好坏，都要求算法为被定罪的罪犯提出判决建议。你甚至依赖算法来保持你的生命，或者至少不让它们杀死你：你的汽车或医疗设备中的控制系统[^1]。

"算法"这个词似乎每天都会出现在新闻中的某个地方。

因此，你需要理解算法，不仅是作为一名计算机科学学生或从业者，还要作为世界的一名公民。一旦你理解了算法，你就可以教育他人了解什么是算法、它们如何运作以及它们的局限性。

本书提供了对现代计算机算法研究的全面介绍。它呈现了许多算法并进行了深入的讲解，同时使它们的设计对所有水平的读者都是易于理解的。所有的分析都已列出，有些简单，有些则更复杂。我们努力保持解释清晰，同时不牺牲覆盖深度或数学严谨性。

每一章都介绍了一个算法、一种设计技术、一个应用领域或相关的主题。算法用英文和伪代码进行描述，伪代码的设计使得任何有一点编程经验的人都能读懂。本书包含 231 幅图表——其中许多包含多个部分——用来说明算法的工作原理。由于我们强调效率是一个设计标准，我们对算法的运行时间进行了仔细的分析。

本书主要供本科生和研究生的算法或数据结构课程使用。因为书中既讨论了算法设计的工程问题，也涵盖了数学方面的内容，所以同样适合技术专业人士进行自学。

在这第四版中，我们再次对全书进行了更新。这些更改涵盖了广泛的内容，包括新的章节和分节、彩色插图，以及我们希望你会觉得更引人入胜的写作风格。

[^1]: 要了解算法影响我们日常生活的众多方式，请参阅 Fry 撰写的书籍[162]。
